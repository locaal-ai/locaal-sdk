Detailed Description

A logging utility class that provides various logging levels and supports custom log callbacks.

The Logger class allows logging messages at different levels (DEBUG, INFO, WARNING, ERROR_LOG). It supports setting a custom log callback function and log level. It also provides a stream-based logging interface through the nested LogStream class.
